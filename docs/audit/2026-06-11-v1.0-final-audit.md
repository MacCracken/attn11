# Audit — v1.0.0 final cut, 2026-06-11

**Scope**: the whole frozen codebase at v0.9.0, as the gate for **v1.0.0** —
the first non-prerelease tag. **Method**: a holistic adversarial pass, five
independent dimensions each rendering an explicit **go / go-with-fixes / no-go**
disposition for shipping 1.0. This is the seventh audit; it builds on the M2/M3
(persistence/SIMD), M5 (AGNOS), M6 (KV/GQA), M7 (BPE/v3), and M8 (security
sweep) audits plus the M10 freeze sweep.

## Verdict: GO — 4 go, 1 go-with-fixes, 0 no-go, 0 blockers

## Findings → dispositions (9; none a blocker)

**Fixed in the 1.0 cut:**
- `--save` failure printed an error but exited 0 → now **exits non-zero** (a
  frozen CLI's exit codes are part of its contract).
- `scripts/version-bump.sh` didn't update `src/main.cyr` `CFG_VERSION` (which CI
  hard-gates) → it now rewrites it, so a bump can't leave `--version` stale.
- SECURITY.md threat model said "once a future release adds file/stdin corpus
  loading" — that shipped in 0.3.0 → reworded to present tense.

**Surfaced for the maintainer (out of this tree / a judgment call):**
- `ai-ml-frontier-2026-expanded.docx` (30 KB binary) is tracked in HEAD and
  would ship in the source tarball — `git rm` it if it isn't meant to ship.
- The SECURITY.md contact (`cyriusmaccken@gmail.com`) differs from the git
  identity (`robert.maccracken@gmail.com`) — confirm it's a live, monitored
  inbox (looks like a possible typo) before tagging.
- Toolchain: the repo pins cyrius **6.1.37** (deliberate, stable v1.0 pin; CI
  installs it from the pin and builds clean). The local wrapper has since
  auto-advanced to 6.1.38 — cosmetic, gitignored `lib/`, does not affect CI.

**Accepted residuals (notes, no action — safe by construction):**
- Duplicate base-vocab bytes in a hand-crafted v3 checkpoint yield a
  degenerate-but-**safe** mapping (the loader's bounds hold); a uniqueness pass
  is optional hardening, not required.
- `file_open` errno (e.g. `ELOOP` = -40) is propagated distinct from the
  loader's validation codes — cosmetic, no safety impact.
